Die Nerdakademie ist eine Arbeitsgemeinschaft der Fachschaft Wirtschaftsinformatik und beschäftigt sich mit sämtlichen Themen aus Soft- und Hardware.

Im Mittelpunkt stehen regelmäßige Treffen, die ein Arbeiten an gemeinsamen oder individuellen Projekten in netter Umgebung ermöglichen und auf denen Workshops und/oder Talks veranstaltet werden. Willkommen sind alle Studenten, unabhängig vom Studiengang, allerdings sollte ein Intresse an IT-Themen vorhanden sein.

Themen/Inhalte:
  • News aus (Computer) Hard/Software diskutieren
  • OSS (Open Source Software) / Linux / UNIX / Windows / Mac
  • Software/Hardware Projekte ausarbeiten
  • Server Administration (Windows, Linux, BSD)
  • Security Analyse (Netzwerk, Verschlüsselung etc.)
  • Workshops, Talks und Podcasts

Wir benutzen Rocket.Chat zur Kommunikation.
